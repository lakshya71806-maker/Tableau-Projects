### Overview
An interactive Tableau workbook built on multi-source loan and sales 
transaction data, featuring a fully designed Story and Dashboard with 
15+ worksheets covering a wide range of chart types and Tableau concepts.

### Charts & Sheets
- **Bar Chart** – Region-wise loan distribution across major U.S. states
- **Line Chart** – Loan amount trends (2014–2019) vs. average interest rates
- **Tree Map** – City-level loan breakdown (e.g. NYC: $4.55M, LA: $3.44M)
- **Scatter Plot** – Sales vs. Profit relationship
- **Pie Chart, Area Chart, Histogram, Dual Combination Chart**
- **Dashboard** – All key visuals in one consolidated view
- **Story** – Guided narrative tying all sheets together

### Concepts Applied
Calculated Fields, Table Calculations, Parameters, Sets,
Custom Territories, Filtering, Sorting, Custom Background

### Data Sources
Loan Transactions, Branch, Customer, Product, and Sales CSVs

### How to View
Download the `.twbx` file and open it in
[Tableau Desktop Public Edition](https://www.tableau.com/products/public)
(free to download).
